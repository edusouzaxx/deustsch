Aqui estão os **verbos que aparecem nas frases** e suas **conjugações no presente (Präsens) 

---

## 1. **heißen** (chamar-se)

| Pronome   | Conjugação |
| --------- | ---------- |
| ich       | heiße      |
| du        | heißt      |
| er/sie/es | heißt      |
| wir       | heißen     |
| ihr       | heißt      |
| Sie       | heißen     |

**Exemplo:**
Ich heiße Eduardo.

---

## 2. **sein** (ser / estar) ⚠️ irregular

| Pronome   | Conjugação |
| --------- | ---------- |
| ich       | bin        |
| du        | bist       |
| er/sie/es | ist        |
| wir       | sind       |
| ihr       | seid       |
| Sie       | sind       |

**Exemplos:**
Ich bin Brasilianer.
Sind Sie Brasilianer?
