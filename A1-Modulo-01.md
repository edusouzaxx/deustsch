# 🔤 1. Alfabeto + Pronúncia

## 📚 Alfabeto alemão

A maioria das letras é parecida com o português, mas a pronúncia muda:

| Letra | Pronúncia aproximada  |
| ----- | --------------------- |
| A     | “a” (aberto)          |
| E     | “ê”                   |
| I     | “i”                   |
| O     | “ô”                   |
| U     | “u”                   |
| J     | “y” (ex: *ja* = “ya”) |
| V     | “f” (ex: *Vater*)     |
| W     | “v”                   |
| Z     | “ts”                  |
| ß     | “ss”                  |

---

## 🔊 Sons MUITO importantes

### 1. “ch”

* **ich** → som leve (quase “rr” suave)
* **Bach** → som mais forte (garganta)

👉 Treino:

* ich (eu)
* nicht (não)
* machen (fazer)

---

### 2. “r”

* Som gutural (na garganta)
* Ex: **rot**, **sprechen**

👉 Não precisa exagerar — apenas não é o “r” brasileiro

---

### 3. Umlaut (ä, ö, ü)

| Letra | Como pensar         |
| ----- | ------------------- |
| ä     | entre “é” e “ê”     |
| ö     | “e” com boca de “o” |
| ü     | “i” com boca de “u” |

👉 Treino:

* fünf (5)
* schön (bonito)
* Mädchen (menina)

# ⚙️ 3. Verbos ESSENCIAIS

## 🔹 Verbo **sein** (ser/estar)

| Pessoa    | Forma |
| --------- | ----- |
| ich       | bin   |
| du        | bist  |
| er/sie/es | ist   |
| wir       | sind  |
| ihr       | seid  |
| Sie       | sind  |

👉 Exemplos:

* Ich bin Eduardo
* Ich bin Brasilianer
* Ich bin müde

---

## 🔹 Verbo **haben** (ter)

| Pessoa    | Forma |
| --------- | ----- |
| ich       | habe  |
| du        | hast  |
| er/sie/es | hat   |

👉 Exemplos:

* Ich habe ein Auto
* Ich habe Zeit

---

# 🧱 4. Estrutura da Frase

## 📌 Regra principal:

👉 **Verbo sempre na posição 2**

### ✔️ Correto:

* Ich **bin** Eduardo
* Ich **wohne** in Basel

### ❌ Errado:

* Ich Eduardo bin ❌

---

# ❓ 5. Perguntas

## 🔹 Perguntas com W

| Palavra | Significado |
| ------- | ----------- |
| Wie     | como        |
| Wo      | onde        |
| Was     | o que       |
| Wer     | quem        |

👉 Exemplos:

* Wie heißen Sie?
* Wo wohnen Sie?
* Was machen Sie?

---

## 🔹 Perguntas SIM/NÃO

👉 Verbo na frente:

* Sind Sie Eduardo?
* Haben Sie Zeit?
